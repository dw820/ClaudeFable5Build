How to read the three pieces you asked about
Autonomous session = agent/. Claude owns the build→render→grade→fix cycle; the harness owns caps + best-so-far. The same primitive (fan-out) also powers preprocess + the grading panel.
Daytona = where that controller + Opus 4.8 + ffmpeg + Remotion + file-memory all live on one filesystem. The agent's persistent home.
Supabase = the decoupling bus. Controller writes events/grades; the browser subscribes via Realtime. Neither side blocks the other — that's what makes the live scorecard reconnection-safe.
The split that matters: blue steps = agent() (judgment); ink steps = plain JS (render, validate, caps). Termination is trustworthy because the caps are code (maxTurns/maxBudgetUsd/wall-clock), even though the agent drives.
